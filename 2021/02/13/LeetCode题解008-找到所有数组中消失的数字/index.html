Ymkiux

发布于：2021年2月13日

找到所有数组中消失的数字

题目

题目
给定一个范围在 1 ≤ a[i] ≤ n ( n = 数组大小 ) 的 整型数组，数组中的元素一些出现了两次，另一些只出现一次。
找到所有在 [1, n] 范围之间没有出现在数组中的数字。
您能在不使用额外空间且时间复杂度为O(n)的情况下完成这个任务吗? 你可以假定返回的数组不算在额外空间内。

示例
示例 示例1
输入 [4,3,2,7,8,2,3,1]
输出 [5,6]
解法

解法

首先根据题目给定一个范围，第一排除空数组,将原数组添加至hashset数组中，根据原数组的长度，判断当hashset数组中不存在遍历值时，将其至添加至新数组中，遍历完返回新数组即可
